How to play

  • You control the python with WASD keys
  • Python will teleport to the opposite side if going through wall
  • Python will die if crash its own tail
  • There are max of 5 foods can appear on the map, each food with disappear after 5 seconds if you don't eat them, new food will appear every 3 seconds
  • Your goal is getting as much foods as possible
  • Good luck!
